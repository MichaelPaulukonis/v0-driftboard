# Feature: Search and Filter in Archived Views

**Description:**
Implement search and filtering capabilities within the "Done," "Archived," and "Deleted" modal views.

**Potential Filters:**
- Search by card title or content
- Filter by date range (e.g., archived this month)
- Filter by user (if applicable in the future)

**Use Case:**
Allows users to quickly find specific past work without manually scrolling through long lists of non-active cards.
